# 👋 안녕하세요, 데이터 중심의 웹 백엔드 엔지니어를 꿈꾸는 김지웅입니다.

현재 대학교 학부 과정 중 **빅데이터 트랙**과 **웹공학 트랙**을 밟고 있으며, 최근 휴학을 하고 **Core CS 기본기**, **실전 백엔드 인프라**, 그리고 **자바스크립트 기반의 구현 능력**을 체계적으로 마스터하는 데 전력을 다하고 있습니다.

> 🎯 **현재의 몰입 목표**
> 대용량 데이터를 안정적으로 처리하고 유저에게 가치 있게 전달하는 "**데이터 특화 백엔드 개발자**"가 되기 위해, 최근 주력 언어를 `Python`에서 `JavaScript(Node.js)`로 확장하며 알고리즘 및 웹 백엔드 원리를 맹렬히 학습 중입니다. **네이버 부스트캠프 웹·모바일 트랙**을 향해 달리고 있습니다.
